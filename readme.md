# Welcome to Penn State Group Finder!

🚨 Our app allows users to create and find groups. 🚨

# Stack 📚
Frontend: **ReactJS** 
https://reactjs.org/
Backend: **NodeJS**
https://nodejs.org/en/
Database: **MongoDB**
https://www.mongodb.com/

## Env Example ✖️



    MONGO_URI=
    JWT_SECRET=
    PORT=
    NODE_ENV=



## Installation ⚙️

 - Clone repository
 - Add environment variables

	``
cd client && npm install
`` 
``
npm install 
``


## Run 🎉

``
npm start
``
